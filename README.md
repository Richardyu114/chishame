# 吃啥么（微信小程序 MVP）

一个帮打工人快速决定 **“今天吃什么”** 的微信小程序。

> 核心：先解决“吃什么内容”（饭 / 面 / 蛋白 / 蔬菜），店铺由用户自己选择。

## 产品定位（v0.3）
- 产品名：吃啥么
- 场景：工作日快速决策，不纠结
- 目标：低成本、少功能、界面简洁但有记忆点
- 非目标：不做外卖平台式的附近找店/比价/下单链路

## 当前实现
- 首页卡片（扑克牌式滑动）
  - 今日吃法组合：主食 + 蛋白 + 蔬菜 + 补充
  - 卡片背景图（本地图片资源）
  - 估算热量 + 推荐理由
  - 古人饮食语录（便于分享）
- 随机与决策
  - `今天吃这个`
  - `看天意`
  - `换一组`
  - `换个口味`
- 个性化设置
  - 默认口味（随机/清淡/均衡/下饭/浓香/辛辣/轻食）
  - 忌口标签（海鲜/牛肉/豆制品/鸡蛋/生冷/辛辣）
- 结果页
  - 展示本次吃法结构
  - 支持复制分享文案

## 技术说明
- 纯前端本地逻辑可运行（无云依赖）
- 数据存储：`wx.setStorageSync`
- 随机引擎：基于结构化组合（不是纯瞎抽）
- 图片：本地静态资源（`miniprogram/assets/food`）

## 本地运行（微信开发者工具）
1. 打开微信开发者工具。
2. 导入项目目录：`/home/chishame`。
3. 使用测试 AppID / touristappid 运行。
4. 进入首页后可直接体验“今天吃什么”卡片流。
